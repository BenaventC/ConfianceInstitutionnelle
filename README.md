# Confiance Institutionnelle en  France de 2006 à 2016

Cette étude a pour but pédagogique d'initier à une analyse comparative et longitudinale. On y teste l'impact de l'orientation politique, de l'environnement de vie et du revenu sur la confiance envers les institutions.


Les données sont extraites de la base de l'[European Social Survey](https://www.europeansocialsurvey.org/data/round-index.html), on en retrouve les résultats sur la page [confiance institutionnelle France0616](https://benaventc.github.io/ConfianceInstitutionnelle/anova.html), l'extraction est disponible [sur ce lien](https://drive.google.com/open?id=1mCwzP3nESSvpgXVv7NUFVwrJMnbeXBtS) (attention > 100mo), voir aussi le [codebook](https://benaventc.github.io/ConfianceInstitutionnelle/codebook.html)
